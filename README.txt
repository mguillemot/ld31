An Adventure on My Screen
~ a game made in 48h by Erhune for Ludum Dare 31

Game released on December 8th, 2014.

This needs Unity 4.6 to be compiled (Free should be totally fine).

This work is released under a MIT license. If you like it, any nice word is appreciated! 

This code uses the following open source libraries:
- SQLite
- NetIrc2 (modified to run under Unity; modified version is present in the source tree)
